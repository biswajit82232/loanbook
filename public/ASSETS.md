# Public assets (required for production)

These PNG files must be present when deploying:

| File | Use |
|------|-----|
| `favicon.png` | Browser tab (48×48) |
| `apple-touch-icon.png` | iOS home screen (180×180) |
| `icon-192.png` | PWA / sidebar / login (192×192) |
| `icon-512.png` | PWA install / maskable (512×512) |

Regenerate from your app store icon set if needed (see project history: `appstore-images` folder).
